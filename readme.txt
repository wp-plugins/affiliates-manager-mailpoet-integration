An addon for the Affiliates Manager plugin to signup the affiliates to your MailPoet Newsletter list

== Description ==

This addon allows you to specify a MailPoet Newsletter list name in the addon settings. 

When affiliates join your site, they get signed up to the specified MailPoet Newsletter list automatically.

This addon requires the [Affiliates Manager Plugin](https://wordpress.org/plugins/affiliates-manager/).

After you install this addon, go to the MailPoet Newsletter settings interface and specify your MailPoet Newsletter list ID and save the settings.

Read the following page for step by step usage documentation:
https://wpaffiliatemanager.com/sign-affiliates-to-mailpoet-list/

== Installation ==

1. Upload `affiliatemgr-mailpoet-addon` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
